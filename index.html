Input: n (the number of lottery tickets)
Create a list of ids from 1 to n
While the length of the list is greater than 1, repeat steps 3-5
Remove every alternate id starting from the first id
If the length of the list is 1, break out of the loop and return the remaining id as the winner
Remove every alternate id starting from the last id
If the length of the list is 1, break out of the loop and return the remaining id as the winner
Repeat steps 3-6 until only one id remains
